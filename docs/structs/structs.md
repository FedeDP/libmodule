# Data structures

Structs API denotes symbols exposed by `libmodule_structs.so`, that expose multiple data structures.  
It can be found in `$includedir/module/structs/`.  
It is made up of multiple separate headers:
* `<module/bst.h>`
* `<module/list.h>`
* `<module/map.h>`
* `<module/queue.h>`
* `<module/stack.h>`
* `<module/itr.h>`
